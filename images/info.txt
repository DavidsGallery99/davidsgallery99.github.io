4. Workflow Summary

Take or edit photos in full quality.

Save originals (resized if needed) into images/large/.

Create smaller versions (same filenames) and save into images/thumbs/.

Example:
images/large/city-1.jpg
images/thumbs/city-1.jpg

Update file names in index.html if you add or remove photos.
